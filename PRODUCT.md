# Product

## Register

product

## Users

Content creators (YouTubers, social media managers) who need to quickly repurpose long-form video content into short-form clips for platforms like TikTok, Instagram Reels, and YouTube Shorts.

## Product Purpose

AICR is an AI-powered tool that automates the video-to-shorts conversion process. Users paste a YouTube URL, and the system uses Klap AI to identify engaging moments, generate clips, and deliver embed-ready short videos.

Success looks like: frictionless workflow from URL paste → clip delivery, with authenticated users getting embed URLs for easy viewing.

## Brand Personality

Clean, efficient, minimal. No fluff, no decorative elements. Every interaction should feel like a tool built for experts who value their time.

## Anti-references

Generic SaaS patterns to avoid:
- Blue/purple gradient hero sections
- Identical card grids with icon + heading + text
- Glassmorphism as default decor
- Hero-metric templates (big number, small label, gradient accent)
- "Welcome to [product]" onboarding with illustration
- Bottom-aligned sticky footers with multiple CTAs

This should feel like a developer's tool, not a marketing SaaS.

## Design Principles

1. **Tool-like confidence** — UI should feel like a precision instrument. Users should trust it works.
2. **Progressive disclosure** — Show complexity only when needed. Default to simplicity.
3. **Show, don't tell** — Let results speak. Embed previews, clip cards, status indicators rather than explanatory text.
4. **Respect attention** — No unnecessary animations, no onboarding tours, no empty states that demand attention.
5. **Form follows function** — Every component earns its place. Remove anything that doesn't serve a task.

## Accessibility & Inclusion

- WCAG 2.1 AA compliance target
- Support reduced motion preferences
- Color-blind safe palette (no red/green as sole differentiators)
- Keyboard navigable for all core workflows
